Answers to Questions

  1. This API is the chuck Noris API. It gives the user completely true facts about chuck norris.
  2. API Website: https://api.chucknorris.io
  3. API Endpoint: https://api.chucknorris.io/jokes/random
  4. There were several options for this API. On this page I use the 'jokes/random' version. However, there are several other options. For example, you can get a list of categories, search jokes by category and get jokes by keywords.
  5. One application where this API would be useful is a website about Chuck Norris. Another use for this api is a website with jokes on it.
